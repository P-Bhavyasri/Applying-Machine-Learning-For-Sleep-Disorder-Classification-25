# Applying-Machine-Learning-For-Sleep-Disorder-Classification-25
“I developed a Sleep Disorder Classification system using Machine Learning and Django. The model analyzes sleep health data and predicts disorders such as insomnia and sleep apnea. I implemented algorithms like Decision Tree, KNN, Random Forest, and SVM, and deployed the project with a web interface. The source code is maintained on GitHub for version control.”


“First I collected the sleep health dataset, then performed preprocessing using Pandas and NumPy. After that I trained multiple machine learning models like Decision Tree, KNN, Random Forest and SVM using Scikit-learn. Finally, I integrated the best model into a Django web application that predicts sleep disorders based on user input.”

Applying Machine Learning for Sleep Disorder Classification

1️⃣ Data Collection

First, the dataset Sleep Health and Lifestyle Dataset is collected.
It contains attributes like:

Age
Sleep Duration
Stress Level
BMI
Heart Rate
Physical Activity

2️⃣ Data Preprocessing
The dataset is cleaned and prepared using:

Pandas
NumPy
Tasks performed:
Removing missing values
Balancing dataset
Feature selection

3️⃣ Model Training

Different machine learning models are trained using Scikit-learn.
Algorithms used:
Decision Tree
K-Nearest Neighbors (KNN)
Random Forest
Support Vector Machine (SVM)

4️⃣ Model Evaluation
The performance of each model is compared using:

Accuracy
Confusion Matrix
Performance graphs

5️⃣ Model Integration
The best performing model is integrated into a web application using the Django framework.

6️⃣ User Interface

A web interface is developed where:
Users enter sleep-related data
The system processes the input
The trained model predicts the sleep disorder

7️⃣ Prediction Output

The system displays the predicted result:
Insomnia
Sleep Apnea
Normal Sleep

This project uses machine learning to classify sleep disorders based on sleep health and lifestyle data such as sleep duration, stress level, heart rate, and BMI. The dataset is processed using Python libraries like Pandas and Scikit-learn, and algorithms such as Decision Tree, KNN, Random Forest, and SVM are used for prediction. The system is implemented as a web application using the Django framework to predict disorders like insomnia, sleep apnea, or normal sleep.
